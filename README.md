# C# Basics Exercises

## Overview
This repository contains a series of **beginner-level C# exercises**.  
The exercises cover fundamental programming concepts including:

- Console input and output
- Arithmetic operations
- Conditional statements (`if`, `else`)
- Loops (`for`, `while`)
- Random number generation
- Basic micro-exercises to practice logic

All exercises are implemented in **console applications** for simplicity and learning purposes.

---

## Exercises Included

### Easy Exercises
1. **Hello World** – Display a welcome message.
2. **Even or Odd** – Determine if a number is even or odd.
3. **Simple Calculator** – Perform addition, subtraction, multiplication, and division.
4. **Guess the Number** – Random number game between 0 and 100.
5. **Multiplication Table** – Generate a multiplication table for a given number.

### Micro Exercises
- Simple input/output exercises.
- Addition of two numbers.
- Checking if a number is even or odd.

### Conditional Exercises
- Compare two numbers and determine which is bigger.
- Determine if a number is positive, negative, or zero.
- Assign grades based on score ranges.

### Loop Exercises
- Print numbers from 0 to 10.
- Calculate the sum of numbers from 0 to 100.
